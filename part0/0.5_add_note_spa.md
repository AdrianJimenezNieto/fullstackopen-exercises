# Sequence diagram of the SPA usage

```mermaid
sequenceDiagram
    participant BROWSER
    participant SERVER

    BROWSER->>SERVER: HTTP GET "https://studies.cs.helsinki.fi/exampleapp/spa"
    activate SERVER
    SERVER-->>BROWSER: 200 OK + HTML code
    deactivate SERVER

    BROWSER->>SERVER: HTTP GET "https://studies.cs.helsinki.fi/exampleapp/main.css"
    activate SERVER
    SERVER-->>BROWSER: 200 OK + CSS code
    deactivate SERVER

    Note right of BROWSER: The browser first asks for the css code because is the first
    code he found on the HTML code


    BROWSER->>SERVER: HTTP GET "https://studies.cs.helsinki.fi/exampleapp/spa.js"
    activate SERVER
    SERVER-->>BROWSER: 200 OK + JavaScript code
    deactivate SERVER

    Note right of BROWSER: Then the browser asks for the JavaScript code

    BROWSER->>SERVER: HTTP GET "https://studies.cs.helsinki.fi/exampleapp/data.json"
    activate SERVER
    SERVER-->>BROWSER: 200 OK + JavaScript code
    deactivate SERVER

    Note right of BROWSER: Finally the browser needs the data.json that the JavaScript code 
    needs to print the information
```
